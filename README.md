# Resume Analyzer

A comprehensive application for analyzing, evaluating, and improving resumes using AI and ATS checking.

## Features

- **Resume Upload**: Support for PDF, DOCX, and text formats
- **ATS Scoring**: Check resume compatibility with Applicant Tracking Systems
- **AI Feedback**: Get personalized suggestions for resume improvement
- **Job Description Matching**: Match your resume against job postings
- **Analytics Dashboard**: View detailed analytics and insights

## Project Structure

- `frontend/` - Streamlit UI components and pages
- `backend/` - Core processing logic
- `database/` - Database models and schema
- `tests/` - Unit and integration tests
- `deployment/` - Docker and deployment configurations

## Installation

```bash
pip install -r requirements.txt
```

## Running the Application

```bash
streamlit run app.py
```
